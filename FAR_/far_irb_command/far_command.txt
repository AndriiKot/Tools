Far cmd:

cls.         очистить экран

irb          вход в интерактивный интерпретатор irb
quit         in irb    выход из интерпретатора irb в ком. строку Far manager
exit         in irb    выход из интерпретатора irb в ком. строку Far manager

fn + end              перевод каретки в конец строки
fn + home             перевод каретки в начало строки

cmd          вход в ком. строку windows
exit         выход из ком. строки в windows и возврат  в ком. строку Far manager
             !!! команда quit  в командной строке не работает !!!
Ctrl + O    выход из Far-cmd и возврат в Far manager
Ctrl + P    Деления экрана на две равные части одна отображает текущею директорию а вторая
            командную строку

Ctrl + UP or DOWN
            Деление области экрана по вертикали с командной строкой

Ctrl + .    запись буква R  !!!

Far manager:

alt + 1      открывает меню поиска
shift + F4   создать файл
F2           сохранение файла
Esc          выход из файла
Enter        управление передастся операционной системе и она попытается запустить или открыть файл
Shift + Enter         откроет файл если папка то передаст управление проводник


F7                    создание папки
F7 styles/style       создает папку styles в которой создает папку style
F6           переместить файл или папки
F6 (../)     переместит попку или в файл в родительскую директорию
shift + F6   переименовать файл или папку
F8           удалить папку или файл


(Ctrl + alt + inset)               копирование  абсолютного пути
                                     F6 при вставки(Ctrl + V) автоматически будет заменен.

F5                скопировать файл или папку
F5 + nameFolder   скопировать данный файл в указанною папку(имя файла можно не указывать)
Enter             открытие данного каталога
F8                удалить папку или файл

F3           просмотр файла
F3 + F4      просмотр файла в 16-ном формате
		0D  символ возврата каретки
		0A  символ конца строки



Ctrl + O     переход к полноэкранной командной строке
             повторное нажатие возвращает в Far manager
Ctrl + P     Деление э
Tab          переход на правую вкладку



Shift + Enter         откроет файл если папка то передаст управление проводнику
Shift + F8            при нахождении в файле открывает список кодировок

Ctrl + alt + insert   копирует путь 
Ctrl + alt + up       перемещение в вверх по командной строке

Alt + F1 , 0         показывает запущенные процессы
