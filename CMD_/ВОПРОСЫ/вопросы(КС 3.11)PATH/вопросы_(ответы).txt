
Команды бывают внутрение и внешние
внутрение команды встроиные в нутрению оболочку CMD
внешние команды не входят в обочоку CMD, они постовляются в виде отдельного
исполняемого файла но этими командами можоно взаимодействовать через командную строку
В основном внешние команды расположены в диске С:\Windows\System32\
Path - это переменная окружения 'Путь к командам'

cmd:
path  выводит список переменного окружения
      выводить адреса всех корзин куда командная строка обращается для поикса исполняемого файла
