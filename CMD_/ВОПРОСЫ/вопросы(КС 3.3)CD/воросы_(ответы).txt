1. Что-бы перейти из текущего положение в корень диска .C: в водим команду 
   cd c:\  
   CD C:\

2. cmd не чуствительна к регистру

3. Что-бы перейти в католог demo folder нужно прописать команду
   cd "demo folder"
   Обезательно в двойных кавычках т.к название пути содержит пробел,
   если вместо двойных кавычек указать одинарные то cmd выдаст ошибку.

4. cd расшивровываетс как Change directory, что переводится как изменить каталог

5. cd + пробел + Tab можно пробежатся по текущим катологам и файлам
   Примечание:
       если исаользовать cmd которая встроенная в far maneger то клавиша Tab
       не даёт должный результат, для этого можно использовать в far maneger ком. cmd
       которая запустет основную командную строку в far maneger. Для того что-бы вернутся
       в far maneger в его превычную ком. стр. нужно ввестри ком.   exit

6. Если в cmd  вести ком. cd пробел а затем первую букву интересеещего вам каталого а затем нажать
   клавишу Tab то будет поставлено первое совподение если такие название есть в данном каталоге

7. Команда cd пробел + Tab          выводит команды сверху вниз
           cd пробел + Tab + Shift  выводит команды снизу вверх

8. cd ..      переход на уровень выше
   cd ..\..   переход на два уровня вверх(со стандартным слешеь)
   cd ../..   может сработай и такой слеш 
   
9. cls        очистит консоль от рание введенных команд.  (работает и Far)
10. Команда CHDIR позволяет записывать путь с пробелами не беря его в кавычки.
11. CD /D [диск:][путь]
    CHDIR /D [диск:][путь]         смена диска и переход в каталог который находится на этом диске
