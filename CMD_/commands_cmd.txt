Вход в cmd:
win                переводит курсор на поиск и открывает меню
win + R		       в виндовс открывает поисковую строку в которой может набирать имена 
                   интересующих нас файлов, папок или программ.
cmd + Enter        открывает командную строку
powershel + Enter  в поисковике открывает программу PowerShell     
echo %USERNAME%    Выводит имя текущего пользователя   

Работа с cmd:
cd      [путь]     смена каталога
cd ..              переместится на уровень вверх     
cls                Очистить экран
Cntrl +  вверх     перемещение экрана вверх
cd                 змена католога  cd c:\
Tab                выводит папку или файл при каждом нажатии сверху вниз
Shift + Tab        выводит папку или файл при каждом нажатии снизу вверх
exit               выход из командной строки
Program Files      что-бы  обратится  к такому каталогу  пищим в cmd такую команду "c:\program files"
NUl > nameFale.js  создает файл    nameFale с расширение js
dir                просмотр всех файлов и каталогов в данной деректории
fn + end           перемещение  курсора в конец строки
fn + home          перемещение  курсора в начало строки
F7                 выводить окно с ведёнными раними командами
code .             открывает текстовый редактор по умолчанию (Например: vscode)
